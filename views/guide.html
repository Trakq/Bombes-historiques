Introduction

Le site web web conçu ci-présent tient comme objectif d’instruire les utilisateurs et de stimuler le désir d’apprentissage que ceux-ci pourraient éprouver à l’égard des procédés chimiques et physiques expliqués interactivement lors de l’utilisation des fonctionnalités de la page web.
En observant les batailles historiques survenues au courant des dernières décennies, on remarque une augmentation massive du stock d’armes explosives. Il pourrait ainsi sembler important que tous bénéficient de la connaissance relative à l’expansion militaire qui les entoure. De cette manière, vous pourrez, chers utilisateurs, combler votre besoin d’apprentissage en exploitant les fonctionnalités de notre site web au moyen d’une simulation ayant comme thème le lancement de bombes qui auront marqué l’histoire. Notre simulateur vous permettra de comprendre facilement comment fonctionnaient les bombes historiques, selon des procédés chimiques complexes, et quelles étaient les répercussions en temps réel sur l’environnement physique composant les villes et les villages touchés par l’explosion.

Démarrage

Sachant que le projet n’est malheureusement pas disponible en ligne, il est possible de cloner le répo git pour accéder aux fichiers composant le projet. Par la suite, il sera nécessaire à l’utilisateur de télécharger les modules requis présents dans le fichier package.json du projet, ainsi que de télécharger le langage Node.js pour assurer la compilation fonctionnelle du programme. Finalement, à l’aide de la commande node effectuée sur le fichier menu.js, l’utilisateur souhaitant exploiter les fonctionnalités du site pourra le faire localement. Dans le cas ou quelconque programmeur souhaiterait continuer le développement du projet, il lui est possible de cloner le répo git du projet Bombes historiques, distribué publiquement et partagé au lien suivant : https://github.com/Trakq/Bombes-historiques. Par la suite, il lui sera recommandé d’installer l’IDE Webstorm accessible en cliquant sur le lien suivant : https://www.jetbrains.com/webstorm/. Finalement, il serait recommandé au programmeur d’utiliser le langage de programmation Node.js pour avancer le développement du projet.

Utilisation

Le démarrage des fonctionnalités est très simple. À partir de la page d’accueil, il est possible de sélectionner la bombe historique souhaitée présentée sous forme de liste déroulante. Par la suite, une fois celle-ci choisie, il ne reste plus qu’à appuyer à l’endroit sur la map ou l’utilisateur souhaitera observer les répercussions à court terme de la simulation de l’explosion. Pour obtenir les informations en lien avec les réactions chimiques composant l’explosion des bombes historiques, il est possible de cliquer sur le bouton « Accéder aux bombes », envoyant l’utilisateur directement à la page prévue à cet effet.

Options

Depuis la page principale du site web, on remarque deux options distinctes qui redirigent respectivement vers le simulateur de bombes historiques et vers la page d'information des bombes historiques.

Par la suite, sur la page du simulateur de bombes, on note plusieurs options comprises dans le menu de gauche, illustré ci-dessous.

Tout en haut du menu se trouve la liste déroulante affichant toutes les bombes que l’utilisateur peut choisir pour la simulation. En dessous se retrouve la liste déroulante affichant les lieux que l’utilisateur pour choisir comme cible pour la simulation. Finalement, il est possible de cliquer sur le bouton « accéder aux bombes » pour être redirigé vers la page d’information des bombes historiques.

Finalement, comme illustré sur l’image ci-dessous, il est possible, depuis la page d’information des bombes historiques, de cliquer sur les boutons en rouge présents à la table des matières présentée à gauche de l’écran pour se situer dans la page. Il est aussi possible de retourner à la simulation en cliquant sur le bouton « Retour à la map ».
